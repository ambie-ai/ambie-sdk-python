# Security Policy

## Reporting a vulnerability

Please email **security@ambie.ai** with the details. Do not open a public GitHub issue for security reports.

We will acknowledge within 2 business days and aim to ship a fix within 14 days for confirmed issues. Responsible disclosure is rewarded with public credit in the changelog and release notes (with your permission).

## Supported versions

The latest minor release on the current major version line receives security fixes.

## Scope

- This repository: the SDK source code.
- Out of scope: the AMBIE service itself. For service-level security reports, use the same email — we track both.
